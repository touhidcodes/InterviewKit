# MongoDB Q&A

### 1. What is MongoDB?
MongoDB is a NoSQL database that stores data in BSON (binary JSON) format, enabling flexible schema design.

### 2. What is a collection?
A collection is a group of MongoDB documents, similar to a table in relational databases.

### 3. What is a document?
A document is a JSON-like object stored in MongoDB.

### 4. Difference between MongoDB and SQL?
- MongoDB: NoSQL, schema-less, document-based  
- SQL: Structured, table-based, fixed schema  

### 5. What is BSON?
Binary representation of JSON-like documents used internally by MongoDB.

### 6. What is ObjectId?
A unique identifier automatically generated for documents.

### 7. What is replication?
Replication ensures data redundancy and high availability using replica sets.

### 8. What is sharding?
Horizontal scaling method distributing data across multiple servers.

### 9. What is an index?
Indexes improve query performance.

### 10. What is aggregation?
A framework for data processing and transformation.
